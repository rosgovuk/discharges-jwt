Registers of Scotland Pattern Library

Overview

The Registers of Scotland Pattern Library is a collection of web application elements, components and design patterns that implements GDS Design Principles and Atomic Design Methodology.

GOV.UK designs and patterns

Registers of Scotland utilises the following GDS SASS/CSS and JavaScript frameworks:

  • Elements
  • Front-End Toolkit
  • Template
  • Rural Payment Styleguide
  • DWP design patterns

Who would benefit from the Pattern Library?

UI designers/developers, UX designers and anyone who is/are planning of creating web application(s) in Registers of Scotland would benefit from the Pattern Library.

This will give your service consistency with the rest of ROSGOV.UK and give you access to our latest patterns and guidance. It will also make it easy to update your service to match changes to the core site.
