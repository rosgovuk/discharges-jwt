Using the Pattern Library

The Pattern Library uses the Atomic Design methodology by Brad Frost. The idea is to bring associations and relationships of elements, components and patterns when designing our UI systems.

For UI/UX designers

Main objective and goal of a UI/UX designer using the Pattern Library is to create or update new/existing patterns, as well as documenting these to serve a purpose of why and when to use such patterns. For example, a pattern for find address.

For developers

To view the HTML mark-up, click the menu "code" > "show pattern info" option on the top-right corner of the browser. This will show the HTML mark-up and documentation of the pattern.
